Istruzioni per l'avvio: Per avviare il programma si effettuino i seguenti
  - npm install
  - npm start
  - Aprire il browser all'indirizzo http://localhost:4200

Istruzioni per l'utilizzo
  - Cliccando sui nomi è possibile indicare la presenza o meno di un ragazzo
  - Cliccando sui pulsanti in fondo alla pagina è possibile consultare le corse future / passate
  - All'interno della cartella "screenshots" sono disponibili un paio di screenshot di esempio
